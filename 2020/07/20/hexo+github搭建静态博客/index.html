hexo+github搭建静态博客

安装Node.js并添加国内镜像源

安装Node.js

https://link.zhihu.com/?target=https%3A//nodejs.org/dist/v9.11.1/node-v9.11.1-x64.msi

首先点击链接下载，所有选项全部默认。安装完成后，打开cmd，输入node -v和npm -v出现版本号则表示安装成功。

添加国内镜像源

这里添加的是阿里国内的镜像源，打开cmd输入以下命令

npm config set registry https://registry.npm.taobao.org

安装git并注册github账号

安装Git

安装了Git我们才能把本地网页文件上传到GitHub上面，安装过程中所有选项全部默认，在最后添加路径时选择Use Git from the Windows Command Prompt

安装完成后在cmd中输入git –version即可验证是否安装成功。

注册github账号

搜索GitHub，进入官网进行注册。

如上图所示新建一个项目，再输入项目名，项目名后面一定要加上后缀.git.io,最后勾选readme初始化

![](/img/repository’s name.png)

项目创建完成后，点击settings，下拉有一个GitHub Pages,然后choose a theme选择博客的主题，回到GitHub Pages，点击那个链接就可以看到自己的网页了。

安装Hexo

在自己喜欢的地方创建一个文件夹用来放自己的博客文件，我的博客文件存在D:\personal\blog目录下。

在该目录下右键选择Git Bash Here,输入npm i hexo-cli -g安装hexo，会有几个报错无视即可，安装完成后输入hexo -v验证是否安装成功。

然后初始化网站，输入hexo init初始化文件夹，接着输入npm install安装必备组件。

现在本地的网站配置也弄好了，输入hexo g生成静态网页，接着输入hexo s打开本地服务器，然后本地打开http://localhost:4000/，就可以看到我们的博客了，最后ctrl c关闭本地服务器。

连接GitHub与本地

在桌面右键选择git bash,输入一下命令：

git config --global user.name "charlieputhr"

git config --global user.email "2541826931@qq.com"

用户名和邮箱由注册github的信息自行修改。

然后生成SSH key：

ssh-keygen -t rsa -C "2541826931@qq.com"

同样的，邮箱改为自己注册github时的邮箱

然后获取密钥内容：

cat ~/.ssh/id_rsa.pub

将密钥内容复制，打开github，在头像下面点击settings，再点击SSH and GPG keys,新建一个SSH，随意取个名字，将复制的内容粘贴到框中，点击确定保存。

gitbash输入

ssh -T git@github.com如果下面的语句出现了你的用户名，就表示成功了：Hi charlieputhr! You’ve successfully authenticated,but GitHub does not provide shell access.

打开博客根目录下的_config.yml文件，这是博客的配置文件，可以在这里修改与博客相关的信息。

修改最后一行的配置，如下图：

repository修改为自己的github项目地址。

编写并发布文章

在博客根目录下右键打开git bash，安装一个扩展npm i hexo-deployer-git。

然后输入hexo new post "article title"，新建一篇文章。

打开D:\personal\blog\source_posts目录，可以发现下面多了一个.md文件，用来存放文章文件。

编写完markdown文件后，跟目录下git bash中输入hexo g生成静态网页，然后输入hexo s可以本地预览效果，hexo d上传github。接着打开github.io就可以看到发布的文章了。

绑定域名

我们现在默认的域名还是xxx.github.io，如果你也想和我一样弄一个专属域名，首先要购买一个域名，阿里云、腾讯云都可以购买。

以我的腾讯云为例，如图所示需要添加两条解析记录：

然后打开github博客项目，点击settings，拉到下面Custom domain处，填上域名保存就好了。

此时项目根目录应该会出现一个名为CNAME的文件，如果没有打开本地博客/source目录，新建CNAME文件，没有后缀！在文件里面写上你的域名保存。最后在博客根目录下git bash，hexo g、hexo d上传到GitHub上面。
